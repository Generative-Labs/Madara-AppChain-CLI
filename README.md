# Madara-AppChain-CLI
Madara-AppChain-CLI

## Installation

Run the following command to install `madcli`:

```bash
curl -L https://raw.githubusercontent.com/Generative-Labs/Madara-AppChain-CLI/main/madcliup/install | bash
```

## Init

```bash
madcli init
```

## Install

```bash
madcli install
```

## Start

```bash
madcli start
```
